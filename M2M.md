### Лабораторная работа №2
```python
import math

print('Введите три переменных для формулы')
a = int(input())
b = int(input())
c = int(input())

def matix():
    P = a + b + c
    p = P/2
    return math.sqrt(p*(p-a)*(p-b)*(p-c))
print(matix())
```
